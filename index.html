About

Hello! I'm Arafat Sani, a passionate developer and aspiring data scientist based in the bustling city of New York. I hold a Bachelor of Science in Computer Science from the New York Institute of Technology, and I'm currently diving deeper into the fascinating world of data science through my Masters program, set to be completed in May 2026.

My journey in tech has been a thrilling ride, filled with innovative projects and constant learning. I thrive on creating dynamic web applications that not only function flawlessly but also offer an engaging and visually appealing user experience. One of my proudest achievements is the YelpCamp Fullstack Web App, a seamless blend of Node.js, Express.js, and MongoDB, topped with interactive maps and secure user authentication. Another exciting project is my JavaScript-based Music Web App, where users can stream audio and visual files with ease, all wrapped in a beautifully designed interface. The Ping Pong Score Keeper is a fun and interactive application I built using HTML, CSS, and JavaScript, allowing real-time score tracking that’s both intuitive and engaging. My Movie Search App leverages the TVMaze API to deliver real-time search results, demonstrating my knack for integrating advanced APIs and enhancing user experience through efficient JavaScript programming

My academic ventures have also been incredibly rewarding. I developed a 3D low-poly rogue-like game using C#, Blender, and Unity, which featured procedurally generated levels and intricate game mechanics. In the realm of data science, I implemented machine learning models for credit card fraud detection, significantly improving detection accuracy and recall. Additionally, my WEKA Data Mining Project involved analyzing the SuperStore_analytics database, where I utilized machine learning algorithms to uncover insightful patterns and trends.

My skill set spans across various programming languages like C#, JavaScript, R, and Python. I am proficient in front-end development tools such as Bootstrap, HTML, and CSS, and I excel in backend technologies like Node.js and Express.js. My expertise also extends to 3D modeling and game development with Unity and Blender, and I am well-versed in database management with MongoDB and data mining using WEKA.

I am driven by a profound passion for technology and the transformative power of AI. I am constantly seeking new challenges and opportunities to learn, grow, and contribute to the ever-evolving tech landscape. Whether it’s developing a visually stunning web application or diving into complex data sets to uncover hidden insights, I am committed to pushing the boundaries of what’s possible.

Log Cabin

This project showcases a modern and advanced music web application, reminiscent of popular platforms like Spotify. The interface is designed to provide a seamless and engaging user experience, emphasizing easy navigation and aesthetic appeal.

The sidebar navigation includes a vertical menu with sections for Playlist, Last Listened, and Recommended tracks, along with a curated list of songs displayed with thumbnails, titles, and artist names.

The main content display features a highlighted song, "Benab ft. Timal - RS5," accompanied by an artist image and a snippet of the song's lyrics, with interactive buttons for playing the song and following the artist to enhance user engagement.

The popular section, with an eye-catching orange background, lists popular songs and artists. Popular tracks such as "Kif Kif," "Mon Poto," "RS3," and "Humain" are displayed with their respective album covers, while a row of circular profile pictures represents popular artists, allowing users to explore more music from their favorite performers.

The sleek bottom media player shows the currently playing song, "Hasta la vista" by PNL, and includes standard controls for play/pause, track navigation, and a progress bar, providing users with control over their listening experience.

This design aims to deliver a rich and immersive musical journey, blending functionality with a visually appealing layout. It’s an excellent example of how user-centric design principles can enhance the overall digital experience in a music streaming application.
